# **App Name**: DiabeAI

## Core Features:

- Diabetes Risk Prediction: Accepts patient health metrics (pregnancies, glucose, blood pressure, etc.) as input. Runs them through a trained model to assess their likelihood of diabetes. Employs an algorithm stacking of Extra Tree, LightGBM, CatBoost and XGBoost algorithms for increased accuracy.
- Model Performance Dashboard: Presents the performance of the ensemble model, including a graph showcasing each model's accuracy (aiming for 92% accuracy).
- Chatbot Interface: Provides a chatbot powered by a language model tool where users can ask questions about their results and diabetes in general.
- Personalized Health Tips: Uses generative AI to create general health tips tailored to the user's specific results and risk factors. Including personalized diet recommendation and suggestion for daily sugar intake limit

## Style Guidelines:

- Primary color: Dark blue (#303F9F) to create a professional and trustworthy feel.
- Background color: Very dark gray (#121212) to make the content stand out. This also supports prolonged usage in dark rooms.
- Accent color: A vivid orange (#FF9800) for the main call-to-action (CTA) buttons, like 'Run Detection'.
- Body and headline font: 'Inter' for a clean and readable user interface.
- Use a consistent set of line icons throughout the dashboard, focusing on health and data-related themes.
- Implement a dashboard layout with clear sections for data input, prediction results, and chatbot interaction.
- Use subtle transitions and animations to enhance user experience.
